---
title: Master Data – Unit
description: Cara mendaftarkan, mengubah, dan menghapus data Unit di PMS
---

**Unit** digunakan untuk menambahkan data unit yang ada atau digunakan di properti. Master Data Unit berfungsi untuk mendaftarkan, mengubah, atau menghapus unit yang tersedia di properti.

Halaman Unit dapat diakses melalui **Master Data → Unit**, dan menampilkan daftar Unit yang sudah didaftarkan pada sistem (nama Unit). Halaman Unit memiliki empat tombol: **Add New**, **Action**, **Edit**, dan **Delete**.

## Cara Menambahkan Unit Baru

1. Klik tombol **Add New** pada halaman Master Data Unit.
2. Isi form pendaftaran Unit yang muncul setelah tombol **Add New** diklik. Data yang harus dimasukkan:

   | Field | Keterangan |
   |-------|-----------|
   | Name | Diisikan dengan nama Unit |

   :::caution[Wajib Diisi]
   Semua data wajib diisikan pada proses pendaftaran Unit. Proses pendaftaran tidak dapat dilanjutkan jika data masih kosong.
   :::

3. Klik tombol **Submit** untuk menyimpan Unit, atau **Cancel** untuk membatalkan proses pendaftaran.

## Cara Mengubah Data Unit

**Edit Unit** digunakan untuk melakukan perubahan data pada Unit yang sudah didaftarkan sebelumnya. Klik tombol **Edit** pada halaman Master Data Unit.

## Cara Menghapus Data Unit

**Delete Unit** digunakan untuk menghapus data Unit yang sudah didaftarkan sebelumnya. Klik tombol **Delete** pada halaman Master Data Unit.
