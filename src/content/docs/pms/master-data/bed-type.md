---
title: Master Data – Bed Type
description: Cara mendaftarkan, mengedit, dan menghapus data Bed Type di PMS
---

Bed Type digunakan untuk mendaftarkan jenis tempat tidur yang tersedia di sebuah properti. Fitur Master Data Bed Type memungkinkan Anda mendaftarkan, mengubah, atau menghapus pilihan bed type yang digunakan dalam sistem.

Halaman Bed Type dapat diakses melalui **Master Data → Bed Type** dan menampilkan daftar bed type yang sudah terdaftar beserta namanya. Halaman ini memiliki empat tombol operasional: **Add New**, **Action**, **Edit**, dan **Delete**.

## Cara Menambahkan Bed Type Baru

1. Klik tombol **Add New** pada halaman Master Data Bed Type.
2. Lengkapi form pendaftaran dengan field berikut:

| Field | Keterangan |
|-------|-----------|
| Name | Nama bed type |

:::note
Semua field pada form wajib diisi. Proses pendaftaran tidak dapat dilanjutkan jika ada field yang masih kosong.
:::

3. Klik **Submit** untuk menyimpan bed type baru, atau **Cancel** untuk membatalkan pendaftaran.

## Cara Mengedit Bed Type

Klik tombol **Edit** pada halaman Master Data Bed Type untuk mengubah data bed type yang sudah terdaftar sebelumnya.

## Cara Menghapus Bed Type

Klik tombol **Delete** pada halaman Master Data Bed Type untuk menghapus data bed type yang sudah terdaftar.
