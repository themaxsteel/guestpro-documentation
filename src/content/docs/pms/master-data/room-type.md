---
title: Master Data – Room Type
description: Cara mendaftarkan, mengedit, dan menghapus data Room Type di PMS
---

Room Type berfungsi untuk mengelompokkan kamar-kamar dalam sebuah properti. Fitur Master Data Room Type memungkinkan Anda mendaftarkan, mengedit, atau menghapus tipe kamar yang tersedia di properti.

Halaman Room Type dapat diakses melalui **Master Data → Room Type** dan menampilkan daftar tipe kamar yang sudah terdaftar beserta nama room type dan income account-nya. Halaman ini memiliki tombol **Add New**, **Action**, **Edit**, dan **Delete**.

## Cara Menambahkan Room Type Baru

1. Klik tombol **Add New** pada halaman Master Data Room Type.
2. Lengkapi form pendaftaran dengan field berikut (wajib diisi semua):

| Field | Keterangan |
|-------|-----------|
| Name | Nama Room Type |
| Income Account | Akun tempat pendapatan dari room type ini akan dicatat |

3. Klik **Submit** untuk menyimpan Room Type, atau **Cancel** untuk membatalkan.

## Cara Mengedit Room Type

Klik tombol **Edit** pada halaman Master Data Room Type untuk mengubah nama room type atau income account yang sudah terdaftar.

## Cara Menghapus Room Type

Klik tombol **Delete** pada halaman Master Data Room Type untuk menghapus data room type yang sudah terdaftar.
