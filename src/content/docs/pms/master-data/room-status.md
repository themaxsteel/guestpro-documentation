---
title: Master Data – Room Status
description: Cara mendaftarkan, mengedit, dan menghapus Room Status di PMS
---

Fitur Room Status digunakan untuk mengelola kondisi kamar di dalam sistem. Secara default, GuestPro menyediakan beberapa status bawaan, namun properti dapat menyesuaikannya sesuai kebutuhan. Status kamar dapat ter-update secara otomatis berdasarkan aturan sistem yang sudah dikonfigurasi.

Properti dapat menambahkan status baru, mengubah status yang sudah ada, atau menghapus status kustom melalui tombol yang tersedia pada halaman Master Data Room Status.

## Cara Menambahkan Room Status Baru

1. Klik tombol **Add New** pada halaman Master Data Room Status.
2. Lengkapi form pendaftaran dengan field berikut:

| Field | Keterangan |
|-------|-----------|
| Code | Kode identitas Room Status |
| Name | Nama Room Status |
| Color | Warna teks |
| Background Color | Warna latar belakang |

:::note
Semua field wajib diisi. Proses pendaftaran tidak dapat dilanjutkan jika ada field yang kosong.
:::

3. Klik **Submit** untuk menyimpan status baru, atau **Cancel** untuk membatalkan proses.

## Cara Mengedit Room Status

Klik tombol **Edit** pada halaman Master Data Room Status untuk mengubah data room status yang sudah terdaftar sebelumnya.

## Cara Menghapus Room Status

Klik tombol **Delete** pada halaman Master Data Room Status untuk menghapus data room status yang sudah terdaftar sebelumnya.
