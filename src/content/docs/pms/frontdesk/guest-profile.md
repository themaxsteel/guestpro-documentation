---

Guest Profile merupakan halaman yang digunakan untuk melihat dan mengelola profil tamu yang pernah didaftarkan. Halaman Guest Profile dapat diakses melalui menu **Frontdesk → Guest Profile**.

Guest Profile menampilkan informasi **Name**, **Email**, **Country**, dan **Nationality**. Tombol yang tersedia pada halaman ini yaitu **Add New**, **Edit**, **Print**, dan **Action**.

## Cara Menambahkan Guest Profile Baru

1. Klik tombol **Add New** pada halaman Guest Profile.
2. Lengkapi data tamu. Field awal yang wajib diisi adalah **Name**.
3. Keterangan masing-masing field:

| Field | Keterangan |
|-------|-----------|
| Name | Nama Tamu |
| Identity No | Nomor identitas KTP / Paspor atau yang lainnya |
| Upload Foto | Berfungsi untuk mengunggah gambar scan atau foto identitas |
| Country | Diisikan dengan data negara tamu |
| Nationality | Diisikan dengan data kebangsaan tamu |
| Birth Date | Diisikan dengan data tanggal lahir tamu |
| Address | Diisikan dengan alamat tamu |
| Phone | Diisikan dengan data nomor telepon tamu |
| Email | Diisikan dengan data email tamu |
| Notes | Dapat diisikan dengan data lain yang diperlukan tentang tamu |

## Cara Mengedit Guest Profile

Klik tombol **Edit** pada halaman Guest Profile untuk melakukan perubahan atau melengkapi data Guest Profile yang sebelumnya sudah dibuat.

## Cara Mencetak Guest Profile

Klik tombol **Print** untuk mencetak atau menyimpan data pada halaman Guest Profile dalam bentuk PDF.

## Import dan Export Data

- **Action → Upload**: digunakan untuk import Guest Profile dari format Excel yang sudah ada.
- **Action → Export Excel**: digunakan untuk menyimpan data pada Guest Profile ke dalam bentuk Excel.
