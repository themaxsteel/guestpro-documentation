Menambahkan reservasi pada PMS GuestPro bisa dilakukan dengan Create Single atau Group Reservation, tetapi jika reservasi berjumlah belasan, puluhan, bahkan ratusan, akan merepotkan jika input reservasi dilakukan satu per satu. Fitur Import Reservation dapat digunakan untuk menambahkan reservasi ke sistem dengan bantuan file Excel agar lebih mudah.

## Langkah-Langkah Import Reservation

1. Buka menu **FrontDesk → Reservation → Action → Import Reservation**.
2. Klik tombol **Download Sample File** untuk mengunduh contoh format file Excel yang dapat digunakan untuk import.

   :::caution
   File Excel yang tidak sesuai format akan menyebabkan data gagal diimport ke sistem.
   :::

3. Klik **Browse** untuk memilih file Excel data reservasi yang sudah disiapkan.
4. Pastikan data pada file Excel sudah sesuai dengan format upload, lalu klik **Read File**. Sistem akan menampilkan hasil pengecekan data pada tabel **Information Upload**, yang menunjukkan apakah data siap diupload atau masih perlu diperbaiki.
5. Jika seluruh data reservasi sudah berwarna hijau (siap diupload), klik **Submit**. Klik **Cancel** untuk membatalkan proses upload.
6. Jika ada notifikasi data yang perlu diperbaiki, perbaiki data pada file Excel sesuai notifikasi yang muncul, kemudian ulangi langkah-langkah di atas.
