---
title: Frontdesk – Guest List Daily
description: Cara melihat dan mencetak daftar tamu harian melalui menu Guest List Daily
---

Guest List Daily merupakan fitur yang digunakan untuk melihat daftar tamu harian berdasarkan tanggal yang diinginkan (secara *default* data yang ditampilkan berdasarkan tanggal sistem). Guest List Daily dapat dilakukan *filter* berdasarkan status tamu (*Arrival*, *Check In* atau *Check Out*). Guest List Daily dapat diakses melalui menu **Frontdesk → Guest List Daily**.

## Informasi yang Ditampilkan

- **Guest Name**
- **Room**
- **Agent**
- **Arrival Date**
- **Departure Date**
- **Pax**
- **Product Include**
- **Special Request**
- **Status**
- **Remark** (ketika Print Guest List Daily)

## Cara Menggunakan

1. Buka menu **Frontdesk → Guest List Daily**.
2. Gunakan filter tanggal dan status reservasi untuk menampilkan data sesuai kebutuhan.
3. Halaman menampilkan informasi reservasi berupa **Guest Name**, **Room**, **Agent**, **Arrival Date**, **Departure Date**, **Pax**, **Product Include**, **Special Request**, dan **Status Reservasi**.
4. Klik tombol **Print Guest List Daily** untuk mencetak daftar tamu harian. Tampilan cetak memiliki format khusus untuk keperluan pencetakan dokumen tamu harian dan turut menampilkan kolom **Remark**.
