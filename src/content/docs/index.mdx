<Aside type="tip" title="Cara menambah panduan baru">
1. Tambahkan file `.md` atau `.mdx` baru di dalam folder `src/content/docs/`.
2. Daftarkan halaman itu di `astro.config.mjs`, bagian `sidebar:`.
3. Jalankan `npm run dev` untuk preview langsung, atau `npm run build` untuk menghasilkan versi final di folder `dist/`.
</Aside>
